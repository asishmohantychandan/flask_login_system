# Flask Login System

A simple Flask-based login system using session management.

## Features
- User Login
- Session Handling
- Protected Welcome Page
- Logout Functionality
- Redirects using Flask routes

## Technologies Used
- Python
- Flask

## Project Structure

```bash
app.py
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/asishmohantychandan/flask_login_system.git
```

2. Navigate to project folder

```bash
cd flask_login_system
```

3. Create virtual environment

```bash
python -m venv venv
```

4. Activate virtual environment

### Windows

```bash
venv\Scripts\activate
```

5. Install Flask

```bash
pip install flask
```

## Run the Application

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
